# Sistem Rekomendasi Masakan Nusantara

Project recommendation system berbasis Streamlit
menggunakan dataset Kaggle.

## Fitur
- Recommendation system
- TF-IDF
- Cosine similarity
- Favorite recipe
- Dark mode
- Detail resep
- Kategori resep

## Cara Menjalankan

Install library:

pip install -r requirements.txt

Jalankan Streamlit:

streamlit run app.py
